Описание проекта.<br>
Проект представляет собой классический форум с аутентификацией и авторизацией пользователя.
<br>
В ходе реализации были применены следующие технологии:
- Java 14<br>
- Spring Boot 2<br>
- Spring Security<br>
- Spring Data JPA<br>
- PostgreSQL<br>
- Liquibase<br>
